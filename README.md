## Quick setup

```
1. 작성
    1. Content/blog/폴더생성/index.md 생성하여 작성
    2. index.md 파일 최상단
    ---
    title: "title"
    date: "date"
    writeAuthor: "author name"
    description: "description"
    categories: [code]
    comments: true
    ---
    날짜는 "YYYY-MM-DDT07:26:03.284Z" 형식으로 작성할 것
    title, date, writeAuthor, description 작성 필수
    
2. Commit
3. Push 전에 terminal에서 git push -f origin main 진행
4. Push
5. terminal에서 yarn deploy
```
